Description

Esta tabla contiene los 'registros' de las ventas que se realice (por esa razon se incluye idventa), incluye idtipoVenta para que se pueda saber si es un item o un servicio. Ademas se incluyen una relacion con tablas como DetalleVentaItem (donde solo incluye una lista de items de dicho idventa), lo mismo pasa con la tabla DetalleVentaServicio (solo incluye una lista de los servicios de dicho idventa). Finalmente, esta tabla hace relacion con la tabla Venta.
